# Marvel Heroes Portfolio Website

A simple responsive website built using HTML, Bootstrap 5, and minimal JavaScript.

This project recreates the provided sample layout and includes a small interactive feature using JavaScript.

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5 (CDN)
- Vanilla JavaScript

---

## Features

- Responsive layout using Bootstrap grid system
- Card-based design for hero showcase
- Consistent image sizing using custom CSS
- Interactive "About This Page" button using JavaScript toggle

---

## How to Run

1. Open https://wijayasf.github.io/website/ on your browser
